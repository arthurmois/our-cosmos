Program 5 - 6/6/2021

In this program, I created a model of our solar system.


Implementation

Using THREE.js, I mplemented a model system of our solar system, TO SCALE.
All 9 planets have textures and are placed correctly in the environment, geocentricly
The scale is 1:10^-4.2
All calculations using km.

Usability

You can switch between viewing each planet using the buttons below.
Left click to rotate around the planet.
Right click to translate the camera.
Use the scroll wheel to zoom in and out.
